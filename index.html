  • Works as a static site on Azure Static Web Apps, Blob Static Website, or App Service.
  • Register your redirect URI in Entra ID.
  • Grant delegated API permissions that allow calls to your Power Platform environment.

Copilot Studio + Power Platform

Chat with your authenticated data agent

The app uses MSAL in the browser, acquires a delegated access token for the configured scopes, creates a conversation against your agent endpoint, and posts user activities over HTTPS.
